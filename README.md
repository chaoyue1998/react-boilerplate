# react-boilerplate

> A React.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at http://10.16.73.77:8080/webpack-dev-server/index.html
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
